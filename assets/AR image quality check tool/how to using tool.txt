1. 운영체제의 터미널을 엽니다. 
- [Windows] cmd(명령 프롬프트) or Windows PowerShell or Git CMD 등
- [macOS] Terminal 등

2. 터미널에서 arcoreimg 실행파일이 있는 폴더로 이동합니다.

3. 해당 폴더에 AR 이미지 인식률을 확인할 이미지 파일을 놓습니다.

4. 터미널에서 arcoreimg 실행파일을 열어 이미지 인식률을 확인합니다.
- [Windows] arcoreimg for windows eval-img --input_image_path={이미지 이름.확장자}
- [macOS] ./arcoreimg for windows eval-img --input_image_path={이미지 이름.확장자}

(참고) 이미지 품질이 75점 이상인 이미지를 사용하는 걸 Google에서는 추천하고 있습니다.
